Bienvenido a mi blog

Este blog es el proyecto que nos pidió Luar, el profe de Aplicaciones Web. La idea es poner en práctica todo lo que estamos viendo en clase creando una página web personal desde cero. Estoy usando herramientas como Zola, que me permite hacer un sitio rápido y fácil de mantener. También trabajo con HTML y CSS para darle estilo y que quede bien en cualquier dispositivo. El blog tiene varias secciones: Asignaturas, donde hablo un poco sobre las asignaturas que estudio. Sobre mí, para que la gente sepa quién soy. Donde se encuentra informacion con la cual contactarma.
