Aplicaciones web

2019-11-27

La asignatura de Aplicaciones Web forma parte del ciclo de Sistemas Microinformáticos y Redes (SMR) y tiene como objetivo enseñarnos a instalar, configurar y administrar aplicaciones que funcionan a través de un navegador. Además, también tocamos aspectos de desarrollo web básico, lo que nos permite entender cómo se construyen y gestionan los sitios web hoy en día.

Durante el curso he podido trabajar con varias herramientas muy útiles y quiero contar un poco mi experiencia:

  • HTML y CSS Empezamos con HTML y CSS, los lenguajes fundamentales de cualquier página web. HTML nos sirve para estructurar el contenido (títulos, párrafos, imágenes, formularios, etc.), mientras que CSS nos permite darle estilo a esa estructura (colores, márgenes, fuentes, animaciones…). Gracias a esto, entendí cómo se construyen las páginas desde cero, y me ayudó a tener más control sobre el diseño.

  • Zola Luego conocí Zola, un generador de sitios estáticos. Es muy diferente a WordPress, porque no usa bases de datos ni requiere un servidor dinámico. Solo generas tus páginas en HTML y las subes. Zola es rápido, ligero y perfecto para cosas como blogs personales o portafolios. Me gustó mucho porque pude usar Markdown, organizar el sitio en secciones y personalizar plantillas con HTML y CSS.

De hecho, estoy creando mi propia web con Zola, donde tengo varias secciones: asignaturas, sobre mí, y contacto. Es un buen proyecto para practicar lo aprendido y tener algo propio que mostrar.

  • WordPress También trabajamos con WordPress, un sistema de gestión de contenidos (CMS) muy popular. Lo instalamos en local y aprendimos a usar temas, plugins, menús, entradas, páginas y widgets. Es ideal si quieres montar una web sin tocar mucho código, pero tener muchas funciones disponibles rápidamente (como blogs, tiendas online, etc.).
